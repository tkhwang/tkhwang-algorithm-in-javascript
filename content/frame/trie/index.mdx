* 특정 텍스트 순서 비교 시 Trie 를 사용하면 빠르게 비교 가능.
* 간단하게 object 사용해서 문자열 순서대로 노드를 확장하면 됨.

### Trie 구성

```typescript
const trie = {};

let node = trie;
for (const ch of word) {
    if (node[ch] === undefined) node[ch] = {};
    node = node[ch];
}
node["word"] = word;
```

### Trie 검색

```typescript
let node = trie;
for (const ch of word) {
    if (node[ch] === undefined) return false;
    node = node[ch];
}
return !!node["word"];
```
