- `dp[i]`는 `(i)`번째 숫자에서 가장 긴 increasing subsequence 의 길이를 의미
- base case: 숫자 하나는 LIS 1 이다.
- 상태: [0, ..., i, ..., N-1]
- 선택
- 상태 전이 방정식

```
// base case
dp[0][0][...] = base case value

// 상태 전이 진행 
for 상태1 in 상태1의_모든_데이터:
  for 상태2 in 상태2의_모든_데이터:
    for ...:
      dp[상태1][상태2][...] = Math.max(선택1, 선택2, ...)
```          

- 상태 순회 순서는 초기값이 있는 위치와 최종 결과가 있는 위치를 고려하여 결정
