* `gcd` 는 두 수의 최대공약수를 찾는 함수.
* a, b 두 수의 최대공약수는 
* b = 0 이면 a 가 최대공약수.
* b != 0 이면 a, b 의 최대공약수는 b, a % b 의 최대공약수와 같음.
* 이를 재귀적으로 계산하면 됨.

```typescript
const gcd = (a, b) => {
    if (b === 0) return a;
    return gcd(b, a % b);
}
```
